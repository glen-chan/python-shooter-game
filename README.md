# python-shooter-game
A mini space shooter game written in python =)

Most of the images were generated using AI. I used Deep AI Image Generator.

If you want to test the game without running python, just save all the folders in your PC, extract the executable from the .7z file and place it in the same location where you placed images, audio and code folders and run the executable. 

A couple of things to know, when you start the game you will see the basic instructions. The actual game will start once you press Y in your keyboard. 

Have Fun!
